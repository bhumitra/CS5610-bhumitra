Validations Using Angular

This experiment explores the the validation controls provided by angular. Enter invalid entries in the fields given below. The submit button will be disabled by angular and you will be prompted to enter correct values.

Name:
Username is required.

Email:
Email is required. Invalid email address.

How it is done?

Below is the code that is responsible for the validations. The controller validateCtrl binds the value for the username and email and checks them is they are invalid. If invalid, an error is thrown to the user. ng-disabled is either true or false according to the values given by the user. It disables the submit button for inappropriate values.

        <form ng-app="" ng-controller="validateCtrl" name="userForm" novalidate>
        <p>Name:<br>
            <input type="text" name="user" ng-model="user" required>
            <span style="color:red" ng-show="userForm.user.$dirty && userForm.user.$invalid">
                <span ng-show="userForm.user.$error.required">Username is required.</span>
            </span>
        </p>
        <p>Email:<br>
            <input type="email" name="email" ng-model="email" required>
            <span style="color:red" ng-show="userForm.email.$dirty && userForm.email.$invalid">
                <span ng-show="userForm.email.$error.required">Email is required.</span>
                <span ng-show="userForm.email.$error.email">Invalid email address.</span>
            </span>
        </p>
        <p>
            <input type="submit" class="btn btn-success" ng-disabled="userForm.user.$dirty && userForm.user.$invalid || userForm.email.$dirty && userForm.email.$invalid">
        </p>
        <hr />
