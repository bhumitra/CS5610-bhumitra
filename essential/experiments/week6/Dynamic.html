Dynamic filtering using Angular

Type something in the below search box and see the change in the values displayed below. This is done dynamically by Angular

  • {{player.name}} - {{player.club}}

How it is done

We can use dynamic filtering using angular by the fillowing peice of code. We have predefined values and when a user types something in the search box, only those results are shown which contain the string typed by the user. (using the search functionality of angular) 

                        function FilterDataModelController($scope) {
                        var player1 = {
                        name: "Ronaldo",
                        club: "Real Madrid"
                        };

                        var player2 = {
                        name: "Messi",
                        club: "Barcelona"
                        };

                        var player3 = {
                        name: "Ribery",
                        club: "Bayern Munich"
                        };

                        var players = [player1, player2, player3];
                        $scope.players = players;
                        }
