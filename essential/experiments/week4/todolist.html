Using JQuery To-do List

Type something in the box and click add. It will get added to the to-do list. To delete what you added, just select it and hit delete. It will be removed from the list. This is done using JQuery by the following code

            $(document).ready(function () {
                $("#add").click(function () {
                    var task = $("#item").val();
                    $("#listtable").append('<tr><td><span><input type="checkbox" name="itemCheck">' + task + '</span></td></tr>');
                });
            });
            function Delete() {
                var x = confirm("Do you want to delete the selected items?");
                console.log(x);
                $('input[type=checkbox]').each(function () {
                    var sThisVal = (this.checked ? $(this).val() : "");
                    if (sThisVal == 'on' && x == true) {
                        $(this).parents("tr").fadeOut();
                        $(this).fadeOut();
                    }
                });
            }
