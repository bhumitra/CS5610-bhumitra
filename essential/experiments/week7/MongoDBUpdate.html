MongoDB - Remove and Update

We will learn how to perform remove and update operations on mongoDB

Current Data in the database

> db.developer.find().pretty()
{
        "_id" : ObjectId("550e7bbef65d61f3c00916f1"),
        "first" : "Bhumitra",
        "Last" : "Nagar"
}
{
        "_id" : ObjectId("550e7c3ef65d61f3c00916f2"),
        "first" : "Arushi",
        "Last" : "Gangrade"
}
{
        "_id" : ObjectId("550e7c73f65d61f3c00916f3"),
        "first" : "Apple",
        "Last" : "Google"
}
{
        "_id" : ObjectId("550e7f1df65d61f3c00916f4"),
        "first" : "Apple",
        "Last" : "Google2"
}
{
        "_id" : ObjectId("550e7f21f65d61f3c00916f5"),
        "first" : "Apple",
        "Last" : "Google"
}

Removing the data with Last name Google2

 
> db.developer.remove({Last: 'Google2'})
WriteResult({ "nRemoved" : 1 })

Updating the record with last as Google and setting the first for it as Apple2 

> db.developer.update({Last: 'Google'},{$set:{first: 'Apple2'}})
WriteResult({ "nMatched" : 1, "nUpserted" : 0, "nModified" : 1 })
> db.developer.find().pretty()
{
        "_id" : ObjectId("550e7bbef65d61f3c00916f1"),
        "first" : "Bhumitra",
        "Last" : "Nagar"
}
{
        "_id" : ObjectId("550e7c3ef65d61f3c00916f2"),
        "first" : "Arushi",
        "Last" : "Gangrade"
}
{ "_id" : ObjectId("550e7c73f65d61f3c00916f3"), "Last" : "Google2" }
{
        "_id" : ObjectId("550e7f21f65d61f3c00916f5"),
        "first" : "Apple2",
        "Last" : "Google"
}

Adding extra fields to a records with update -- Here we add the field "type" to the record with Last as Google and setting the first and Type for it

> db.developer.update({Last: 'Google'},{$set:{first: 'Apple2',type:'company'}})
WriteResult({ "nMatched" : 1, "nUpserted" : 0, "nModified" : 1 })
