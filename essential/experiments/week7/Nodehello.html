Introduction to Nodejs

Node.js® is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices. We can run javascript on desktop instead of a browser using nodejs.

Installation steps on how to use Nodejs

  • Install node js from its website - Nodejs. Click on install and run the downloaded setup.
  • In the environment variables, edit the path. It should be "C:\Program Files\nodejs". Remember there should be no \ after nodejs in the path or else it wont work.(it was so in my case)
  • create a directory anywhere in your computer. Create a file nodehello.js
  • type "console.log("node hello")" in that file and save it
  • Open power shell
  • Navigate to the path where file is created
  • Run this command on the powershell > node .\nodehello.js
  • Output should be "node hello" on the console

Now install "express"framework using the node package manager

  • Run the following code in the powershell: > npm install express
  • Create a file serverApp.js in the working directory with the following code 
                    var express = require('express')
                var app = express()
                app.get('/', function (req, res) {
                res.send('Hello World')
                })
                app.listen(3000)
    This will create a server app. Whenever a request comes, our app is listening for incoming url's the response "Hello World" sent back.
  • Run this command on the powershell to start the server app we created. > node .\serverApp.js
  • Our server is listening at port 3000. So got to "localhost:3000" on the browser. You will see the Hello World on the browser which our server app has returned.
