MongoDB - Insert and Find

We will learn how to perform insert and find operations on mongoDB

Connecting to DB

> show dbs
local  0.078GB
> use cs5610
switched to db cs5610
>
>
> show collections
> db.developer
cs5610.developer

Insert Json object into the database using the following command

 

> db.developer.insert({first: 'Bhumitra', Last: 'Nagar'})
WriteResult({ "nInserted" : 1 })
>

Finding data in the database -- equivalent to select *

> db.developer.find()
{ "_id" : ObjectId("550e7bbef65d61f3c00916f1"), "first" : "Bhumitra", "Last" : "Nagar" }
> db.developer.insert({first: 'Arushi', Last: 'Gangrade'})
WriteResult({ "nInserted" : 1 })
> db.developer.find()
{ "_id" : ObjectId("550e7bbef65d61f3c00916f1"), "first" : "Bhumitra", "Last" : "Nagar" }
{ "_id" : ObjectId("550e7c3ef65d61f3c00916f2"), "first" : "Arushi", "Last" : "Gangrade" }

Inserting more records into the database

> db.developer.insert({first: 'Apple', Last: 'Google'})
WriteResult({ "nInserted" : 1 })
>

> db.developer.find()
{ "_id" : ObjectId("550e7bbef65d61f3c00916f1"), "first" : "Bhumitra", "Last" : "Nagar" }
{ "_id" : ObjectId("550e7c3ef65d61f3c00916f2"), "first" : "Arushi", "Last" : "Gangrade" }
{ "_id" : ObjectId("550e7c73f65d61f3c00916f3"), "first" : "Apple", "Last" : "Google" }
>

Showing the inserted data into readable format

> db.developer.find().pretty()
{
        "_id" : ObjectId("550e7bbef65d61f3c00916f1"),
        "first" : "Bhumitra",
        "Last" : "Nagar"
}
{
        "_id" : ObjectId("550e7c3ef65d61f3c00916f2"),
        "first" : "Arushi",
        "Last" : "Gangrade"
}
{
        "_id" : ObjectId("550e7c73f65d61f3c00916f3"),
        "first" : "Apple",
        "Last" : "Google"
}

Finding the data according to the given first name.

> db.developer.find({first:'Bhumitra'}).pretty()
{
        "_id" : ObjectId("550e7bbef65d61f3c00916f1"),
        "first" : "Bhumitra",
        "Last" : "Nagar"
}
