Creating a landing page for Project - Tickster

Tickster is a Movie Hub and an Event Booking website

Check the landing page here - Landing Page

The landing page uses jquery and CSS to accomplish the look. As you move your mouse/hover to either of the section, the corresponding section expands. This is achieved by the peice of code given below. The page is also responsive. 

       $(document).ready(function ($) {
            var $left = $('.landing-left');
            var $right = $('.landing-right');
            var time = 0, type = "easeout";
            $left.click(function (event) {
                window.location = '/movies';
            }).hover(function (e) {
                $(this).animate({
                    width: "70%"
                }, time, type)
                $right.animate({
                    width: "30%"
                }, time, type)
            });
            $right.click(function (event) {
                window.location = '/events';
            }).hover(function () {
                $(this).animate({
                    width: "70%"
                }, time, type)
                $left.animate({
                    width: "30%"
                }, time, type)
            });
        });
