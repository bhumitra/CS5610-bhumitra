• Responsive slider - felxslider2

    Implementing the slider to show the images in the form of a tickr. This will be used to display image posters which are clickable. It is also responsive. Links are given at the bottom of the page.

    Explaination:

    This is implemented by using the flex-slider plugin for angular. It can be installed using - bower install angular-flexslider and including its css in the page. Also, AngularJS app, you'll need to import the angular-flexslider module: 

    angular.module('myModule', ['angular-flexslider']);
    needs to be imported

    HTML:

    flex slider has a similar function like the ng-repeat called as slide. Also it provides various options for specifying the animation, number of items in the carousal, etc. 

                 <flex-slider class="carousel" slide="s in slides track by $index" animation="slide" maxitems=10 animation-loop="false" item-width="210" item-margin="5">
                <li>
                    <img ng-src="{{s}}">
                </li>
            </flex-slider>
