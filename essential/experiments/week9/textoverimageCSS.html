Animate text over images on hover without JavaScript

There are three animations happening simultaneously when you move your mouse over each image:

  • A zoom effect on the image itself, created by scaling the image up 140%
  • The text and its transparent black background fading in
  • The text dropping in from the top
This forms the base layout of my project's homepage.

HTML:

Put the images in a list, and then use the HTML5 figure and figcaption elements to hold the images and associated text:

          <ul class="photo-grid">
         <li>
             <a href="http://www.showbiz411.com/wp-content/uploads/2015/04/avengers.jpg">

                     <figcaption><p>Avengers</p></figcaption>
                 </figure>
             </a>
         </li>
         <li>

CSS:

Modify the list style so that our images display side-by-side rather than in a list with bullets. Also,setting width and height for the list items, images, captions and the paragraphs inside the captions. We position the captions over the images, but set their opacity to 0 so that they’re not visible.

   .photo-grid {
	margin: 1em auto;
	max-width: 1106px;
	text-align: center;
   }
 
   .photo-grid li {
	border: 5px solid white;
	display: inline-block;
	margin: 1em;
	width: 289px;
   }
 
   .photo-grid img {
	display: block;
	height: auto;
	max-width: 100%;
    }
 
   .photo-grid figure {
	height: 163px;
	overflow: hidden;
	position: relative;
	width: 289px;
   }
 
   .photo-grid figcaption {
	background: rgba(0,0,0,0.8);
	color: white;
	display: table;
	height: 100%;
	left: 0;
	opacity: 0;
	position: absolute;
	right: 0;
	top: 0;
	z-index: 100;
   }
 
   .photo-grid figcaption p {
	display: table-cell;
	font-size: 1.5em;
	position: relative;
	top: -40px;
	width: 289px;
	vertical-align: middle;
   }

Showing text on hover: setting opacity of the figcaption to 1 when the list item is hovered. Now text is visible

.photo-grid li:hover figcaption {
	opacity: 1;
}

Zoom the images: using animation

    .photo-grid img {
	display: block;
	height: auto;
	-webkit-transition: all 300ms;
	-moz-transition: all 300ms;
	transition: all 300ms;
	max-width: 100%;
    }
 
    .photo-grid li:hover img {
	-webkit-transform: scale(1.4);
	-moz-transform: scale(1.4);
	transform: scale(1.4);
    }

Dropping text effect: adding in the CSS transition and a CSS transform to move the text 40px down.

   .photo-grid figcaption p {
	display: table-cell;
	font-size: 1.5em;
	position: relative;
	top: -40px;
	width: 289px;
	-webkit-transition: all 300ms ease-out;
	-moz-transition: all 300ms ease-out;
	transition: all 300ms ease-out;
	vertical-align: middle;
    }
 
    .photo-grid li:hover figcaption p {
	-moz-transform: translateY(40px);
	-webkit-transform: translateY(40px);
	transform: translateY(40px);
    }

Fading effect: – a transition and a transition-delay 

    .photo-grid figcaption {
	background: rgba(0,0,0,0.8);
	color: white;
	display: table;
	height: 100%;
	left: 0;
	opacity: 0;
	position: absolute;
	right: 0;
	top: 0;
	-webkit-transition: all 300ms;
	-moz-transition: all 300ms;
	transition: all 300ms;
	-webkit-transition-delay: 100ms;
	-moz-transition-delay: 100ms;
	transition-delay: 100ms;
	z-index: 100;
    }
