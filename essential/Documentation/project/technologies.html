TICKSTER- Technologies Used

TICKSTER is developed in MEAN (MongoDb, Express, Angular, Nodejs) stack and is fast and highly responsive.

Following Technologies were used to build this application:

  • Nodejs
    • brypt-nodejs (for encrypting the passwords before storing them in the database)
    • body-parser (for getting parameters from the request)
    • connect-flash (for displaying error messages on the login screen)
    • cookie-parser (for cookies)
    • ejs (Embedded Javascript)
    • express (server)
    • express-session (for sessions)
    • fs (for renaming and deleting profile pictures on the file system)
    • mongoose (for MongoDb queries)
    • multer (for image upload)
    • passport (for authentication)
    • passport-local (for authentication)
  • MongoDb
  • Bootstrap
  • Infinite-Scroll
  • flexslider2
  • AngularJS
    • Angular Animate
    • Angular Route
  • HTML
  • CSS
  • jQuery
    • jQuery Cookie
  • Javascript
