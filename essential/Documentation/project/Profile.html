TICKSTER- Profile Page

TICKSTER's Profile page displays the user's info - Profile Photo, Name, About the User, favorite movies and user followed. There can be two possibilities - User checking his own profile page or User checking some one elses profile. If any of these details is not present it is not displayed.

User Checking Own Profile:

When the user opens his own profile, The text displayed on the page is different. It says "YOUR" favorite movies, "YOU" follow and there is an option to delete his own account

User Other User's Profile:

When the user opens other users profile, The text displayed on the page is different and there is an option to follow that user. If you click on follow, the button changes to red saying unfollow and that user is added to your list of followed users.

Delete Account (Here the user logged in is Arushi):

Click on delete account to delete your account. The user is prompted for confirmation before account is deleted.
