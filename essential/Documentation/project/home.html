TICKSTER- Home Page

TICKSTER's home page layout is simple. It has a navigation bar on the left, a search box on top right cornor, a tickr carousel and a photo grid.

Details:

The content is fetched from Rotten Tomatoes API and displayed as a tickr for Upcoming movies and as a dynamic scroll photo grid for the movies in theatres. All the Images are clickable. Also hovering over the Movies In Theatre images displays the information of the movie. Click on it for more details.

Movies Coming Soon: The homepage displays the upcoming movies tickr in a responsive carousal.
Movies In theaters: The homepage shows the movies in theatres in a photo grid which on hover displays the movie details. It is implemented with INFINITE/dynamic SCROLLING, so there are no pages. As you scroll down, more content is loaded
Search: Search for a movie in the search box. Resuts displayed as a table which is also dynamic/infinite scroll enabled.
Hover: Hovering over the Movies In Theatre images displays the information of the movie - Title, Release Date, Rating and Critic score. Click on any movie for more details.
