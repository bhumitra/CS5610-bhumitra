TICKSTER- Signup Page

TICKSTER's Signup page enables the user to create a new account for TICKSTER so that he can enjoy extended functionality of the application.

Details:

By entering the following details

  • Name
  • Tell us something about yourself
  • Profile picture
  • Email
  • Password
the user can register to tickster. All fields are required and if user forgets to enter any one of them he is prompted. After a successful signup, the user is taken to the login page.
