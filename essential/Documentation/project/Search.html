TICKSTER- Search Page

TICKSTER's Search page shows all matching search results of the movie user has searched for. It implements dynamic scroll. Initially only a part of the search is shown. If number of results is more, and as the user scrolls and reaches the bottom of the page, more results are loaded in the table. The search page shows the Poster, Title, Rating, Audience Rating out of 5 and critics rating %.
