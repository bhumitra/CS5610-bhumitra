TICKSTER- Movie Details Page

TICKSTER's Movie Details page shows movie poster, Title, Release Date, Cast, Directed By, Plot, Genre, Rating, Audience rating, Critic's Rating and Studio.

Features:

You may also like: If a movie has movies similar to it, they are displayed under heading "You may also like". These are recommendations from tickster. If no similar movies are found in the rottentomatoes api, then this area is not shown
Like: If the user is logged in he can like the movie which will be added to his list of favourite movies otherwise the like button is disabled
Showtimes: The showtimes for the movie are fetched from TMS API. It is limited 50 requests per day!. User can search for movie screening in his nearby area by typing in Zip code and Date. Data is fetched and you can proceed with booking on Fandango.
Reviews: If the user is logged in he can post his own review the movie which will be added to the reviews for that movie otherwise review button is disabled.
Edit Review: If the user is logged in he can edit his posted reviews for a movie by clicking the edit button.
